## 🎯 The 4 Steps (Every Time)

### Step 1: MongoDB (Free - 5 min)
```
1. https://www.mongodb.com/cloud/atlas
2. Create account → Create cluster (free)
3. Create user: taskmanager
4. Get connection string
```

### Step 2: Code (2 min)
```
1. backend/.env → Add MONGODB_URI
2. team-task-manager/.env → Add VITE_API_BASE_URL
3. Done!
```

### Step 3: GitHub (1 min)
```
1. git add .
2. git commit -m "Ready for Railway"
3. git push origin main
```

### Step 4: Railway (5 min)
```
1. https://railway.app
2. New Project
3. Deploy from GitHub
4. Select Ethera_AI_Project
5. Click Deploy
6. Wait for both services to start
```

**Total: 13 minutes to live deployment! ⏱️**
