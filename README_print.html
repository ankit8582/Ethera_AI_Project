📋 Project Overview
Name:Ethera AI - Team Task Manager
Version:1.0.0
Description:A premium, full-stack team task management app with real-time collaboration, Firebase authentication, and MongoDB backend.
🛠️ Tech Stack

⚛️ Frontend

React 19
Vite (build tool)
TailwindCSS v4
Custom CSS (Glassmorphism)
React Router DOM
Firebase Authentication

🖥️ Backend

Node.js
Express.js
MongoDB (Atlas)
Mongoose ODM

Railway.app (Deployment)

✨ Key Features
  • User Registration & Login (Firebase)
  • Password Reset via Email
  • Create, Edit, Delete Tasks
  • Task Categories & Priority levels
  • Due Date tracking & Overdue detection
  • Task Completion toggle
  • Drag & Drop task reordering
  • Search & Sort tasks
  • Project & Team Management
  • Add/Remove Team Members
  • Assign Tasks to Members
  • Dashboard Statistics
  • Progress Bar (completion %)
  • Dark / Light Mode toggle
  • Offline-first (LocalStorage)
  • Responsive Design (mobile+desktop)
